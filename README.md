# Tidal-TUI
<img width="1491" height="1064" alt="Screenshot 2026-07-05 at 10-01-33 TIDAL (Edit)" src="https://github.com/user-attachments/assets/1379276f-fd51-4f3a-bdc1-2d02701ccedc" />
A terminal user-interface-like theme for Tidal, based on spicetify-text.


## Installation
### Browser
1. Install the browser extension Stylus. \
[Firefox](https://addons.mozilla.org/en-CA/firefox/addon/styl-us/) | [Chrome (Edge, Opera, Brave etc.)](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
2. Install the theme. \
   [Tidal-TUI](https://userstyles.world/style/27576/tidal-tui)
3. Edit the theme, and adjust the top two variables so that the 
margin between the player bar and the sections above looks correct. You 
are also free to adjust the other variables.


### Windows, MacOS, & Linux Desktop App (TidaLuna)

1. Run the [Luna](https://github.com/Inrixia/TidaLuna) client mod installer. \
[Download Luna installer](https://github.com/jxnxsdev/TidaLuna-Installer/releases)
2. After installing, open Tidal, click your profile, and click on Luna settings.
<img width="273" height="392" alt="image" src="https://github.com/user-attachments/assets/7c5c8f90-cc3a-4d96-b223-f3fcae4c03a3" />

3. Go to the Plugin Store tab.
4. Search for "Themer", and install it.
5. Go to the Themes tab.
6. Copy the link below and paste it into the "Install from URL" text box.
```
https://raw.githubusercontent.com/InterstellarOne/Tidal-TUI/refs/heads/main/tidaluna.css
```
7. Press **Ctrl + E** to open the theme editor.
8. Go to the the file linked below, copy the contents, and paste them into the theme editor. \
https://github.com/InterstellarOne/Tidal-TUI/blob/main/tui-settings.css
9. Adjust the top two variables so that the margin between the player bar and the sections above looks correct. You are also free to adjust the other variables.
